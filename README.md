# Decal

## Installing Decal

```bash
npm install calvient-decal
```

## Developing Decal
To develop, run `yarn storybook`.

## Publishing to Decal
If you have not done so, make sure to run `npm login` first.

We follow semantic versioning rules.

To publish changes, run:

```bash
npm version {major}.{minor}.{patch}
npm publish
```
